\# Yorkshire Industry Employment Analysis



\## Project Overview

This project analyses employment trends across industries in Yorkshire and the Humber from 2015 to 2024, examines the impact of COVID-19, and predicts industry employment in 2030.



\## Repository Contents

\- \*\*R Code\*\*: Scripts for data cleaning, analysis, and visualisation.

\- \*\*Dataset\*\*: `bres\_yorkshire\_industry.xlsx` containing industry employment data.

\- \*\*Plots\*\*: Folder containing all visualisations used in the report.
